# Python Project One

A collection of three fun programs built in Python with a user-friendly menu system.

## Features

This project includes three interactive programs:

1. **Magic Eight Ball**  
   A fortune-telling game where users ask questions and receive random answers.

2. **Fortune Cookie Generator**  
   Generates fortunes along with four random numbers.

3. **Fizz Buzz**  
   A classic programming exercise that prints numbers from 1 to 100, replacing multiples of 3 with "Fizz", multiples of 5 with "Buzz", and multiples of both with "FizzBuzz".

## Requirements

- Python 3.x
- No external dependencies beyond the standard Python library

## How to Run

1. Download `main.py`
2. Make sure you have Python installed on your system
3. Navigate to the directory you saved the file and run the program using the command:
   ```bash
   python3 main.py
   ```

## User Interface

The program features a clean menu system that clears the console between selections. Users can navigate the menu by entering numbers 1-3 for the respective programs, or type "q" to quit.

### Menu Navigation
- Enter "1" for Magic Eight Ball
- Enter "2" for Fortune Cookie
- Enter "3" for Fizz Buzz
- Type "q" to exit the program
